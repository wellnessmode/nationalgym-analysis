# NationalGym Body Scan — 트레이너 사용 매뉴얼

**버전**: v1.14.1 | **최종 수정**: 2026-04-11 | **앱 주소**: https://wellnessmode.github.io/nationalgym-analysis/

---

## 1. 앱 소개

**NationalGym Body Scan**은 NationalGym 전용 체형분석 프로그램입니다.

- MediaPipe 기반 **자동 스켈레톤 감지**
- Claude AI **자세 분석 + 교정운동 처방**
- Before / After **비교 분석**으로 개선 결과 시각화
- **클라우드 저장** (Cloudflare R2)으로 데이터 안전 보관

## 2. 주요 기능

| 기능 | 설명 |
|---|---|
| **영상 캡처** | 셔터 1회 → 1.5초간 15프레임 캡처 → 최적 프레임 자동 선택 |
| **스켈레톤 감지** | 자동 8회 반복 감지 → 최고 점수 결과 사용 |
| **임상 오버레이** | Plumb line · 어깨/골반 각도 · CVA · 편차 mm 자동 표시 |
| **체크리스트 AI 가이드** | 사진 기반 AI가 좌우 비대칭 자동 판정 (검사당 수동 실행) |
| **AI 자세 분석** | Sonnet 4.5 모델 — 서술형 문제점 요약 (검사당 최대 3회, 이전 결과 선택 가능) |
| **수기 검사** | Patrick · Prone · Calf · Neck Palpation 체크리스트 직접 입력 |
| **비교 분석** | Before(빨강) / After(파랑) 오버랩 + AI 자동 싱크 |
| **Clinical Analysis** | 비교 분석 시 개선된 항목 AI 분석 (Sonnet, 최대 3회) |
| **트레이너 리포트** | AI 종합 분석 → 검수/편집 → PDF 생성 → 고객 전송 |
| **클라우드 저장** | 사진 + 체크리스트 + AI 결과를 R2에 압축 저장 |
| **종합 자세 점수** | 0~100점 가중 평균 (5단계: 우수/양호/보통/주의/위험) |
| **세션 히스토리** | 재검사 시 이전 점수와 트렌드 차트 비교 |
| **얼굴 블러** | 공유/PDF 시 얼굴 자동 모자이크 (토글) |
| **자동 백업** | 변경 시마다 + 30초마다 자동 저장. 앱 재시작 시 복원 프롬프트 |

## 5. 트레이너 사용법

### 5.1 시작하기

1. **앱 접속**: https://wellnessmode.github.io/nationalgym-analysis/
2. **비밀번호 입력** (최초 1회): `nationalgym2026`
3. **"체형분석 시작"** 버튼 탭
4. **회원 정보 입력**:
   - 회원 이름 (필수)
   - 담당 트레이너 (드롭다운에서 본인 선택 — 클라우드 저장 시 본인 폴더로 분류됨)
   - 키 (cm) — 입력하면 측정치가 mm 단위로 자동 변환
   - 트레이너 이메일/전화 — 리포트 전송 시 사용

### 5.2 Before 촬영

**검사 순서대로** 스크롤하며 촬영:

1. **사진 검사** (14개): "카메라로 촬영" 또는 "갤러리 업로드" 탭
   - 셔터 누르면 **1.5초 영상 캡처** → 15프레임 중 최적 자동 선택
   - 포즈 가이드(녹색 실루엣)에 맞춰 촬영
   - Device level 게이지(우측)가 **초록색**일 때 촬영 (기기 수직 ±2°)
   - **SLR은 📱 가로로 돌려서 촬영!**

2. **수기 검사** (8개): 노란 배너 "📝 수기 검사" 아래 체크리스트 직접 체크
   - 메모 입력란에 특이사항 기록 가능

3. **AI 기능** (사진 검사에만):
   - **"📋 체크리스트 AI 가이드"**: AI가 좌우 비대칭 자동 판정 → 빨간 테두리로 추천 표시 → 클릭해서 확정
   - **"🧠 AI 자세 분석"**: 해당 검사의 문제점을 2~3문장으로 서술 (최대 3회, ◀▶로 이전 결과 선택)

4. **일괄 저장**: Before 촬영 다 끝나면 맨 아래 **"💾 Before 사진 일괄 저장"** → 공유 시트 → **"이미지 저장"** → 사진첩 저장

### 5.3 운동 지도

Before 촬영 완료 → 고객에게 교정 운동 지도 (Before 사진의 AI 자세 분석 참고)

### 5.4 After 촬영

1. **"After 촬영"** 탭으로 이동 (상단 네비 또는 하단 플로팅 버튼)
2. 수기 검사는 표시 안 됨 (Before에서만 입력)
3. 사진 검사만 다시 촬영
4. 체크리스트 AI 가이드는 After에서도 사용 가능

### 5.5 비교 분석

1. **"비교 분석"** 탭으로 이동
2. 각 검사별 오버랩 카드:
   - **투명도 슬라이더**: Before ↔ After 비율 조절
   - **"AI 자동 싱크"**: Before/After 자동 정렬
   - **"수동 미세 조정"**: 드래그로 위치/크기/회전 조절
   - **마커/측정선**: 보기 모드 → 마커/측정선 모드로 전환해서 추가
3. **Clinical Analysis AI**: "🧠 AI Clinical Analysis 실행" 버튼 → 개선된 항목 표시 (최대 3회)

### 5.6 리포트

1. 비교 분석 맨 아래 **"📋 리포트 받기"** 탭
2. **트레이너 편집 모드** 진입:
   - AI 종합 분석 생성 (최대 3회, ◀▶ 선택)
   - 트레이너 코멘트 직접 작성
   - 교정 운동 처방 확인
   - ☁️ **클라우드에 세션 저장** (필수 — 데이터 보존용)
3. **PDF 전송**:
   - "👁 PDF 미리보기·다운로드" → PDF 확인
   - "📧 고객에게 PDF 전송" → 공유 시트 → 카카오톡/메일 선택

### 5.7 클라우드 저장/불러오기

- **저장**: 리포트 화면 → "☁️ 클라우드에 세션 저장"
- **불러오기**: 랜딩 화면 또는 리포트 화면 → "📥 클라우드에서 불러오기"
- 본인(트레이너) 이름 기준으로 세션이 분류됨 → **본인 세션만 표시**
- ⚠️ **반드시 트레이너 드롭다운에서 본인 이름을 선택**한 후 저장/불러오기

### 5.8 세션 종료 후

1. 다음 고객: **🏠 처음** 버튼 (헤더) → 랜딩 화면 → "체형분석 시작"
2. 이전 세션은 자동 저장되어 있음 (앱 재시작 시 복원 프롬프트)

## 6. 주의사항

### 촬영 시
- ❌ **거울 앞 촬영 금지** (스켈레톤이 거울 속 사람도 감지)
- ❌ 복잡한 배경 금지 (단색 배경 권장)
- ✅ 전신이 프레임 안에 들어오게
- ✅ 밝은 조명
- ✅ 카메라 수직 유지 (Device level 초록색)
- ✅ SLR은 **반드시 가로로 돌려서** 촬영

### 데이터 보호
- 자동 저장: 변경 시마다 + 30초마다 자동 저장
- 앱 재시작 시 "이전 세션 복원?" 프롬프트 표시
- **중요 세션은 반드시 ☁️ 클라우드 저장** (기기 문제 시 복구 가능)
- 얼굴 블러: 회원 정보 아래 🔒 체크 → 공유/PDF 시 얼굴 자동 가림

### AI 기능
- AI 사용량: 헤더에 `AI N/300` 배지로 실시간 확인
- 한도 초과 시: 배지 탭 → 리셋 (확인 후)
- 스켈레톤이 안 잡힐 때: "스켈레톤 재분석" 버튼 반복 (AI 비용 없음)
- 네트워크 오류 시: 포즈 기반 자동 판정으로 대체됨 (정확도 다소 낮음)

| 증상 | 해결 방법 |

| 비밀번호를 까먹었어요 | `nationalgym2026` 입니다 |

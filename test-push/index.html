3. 푸시 수신 대기

4. iOS 설치 절차

  1. 이 페이지를 Safari로 접속 (Chrome/Firefox iOS 안 됨)
  2. 하단 공유 버튼(⬆️) → "홈 화면에 추가"
  3. 홈 화면 아이콘 탭 → 다시 진입 (반드시 standalone)
  4. 위 1번 권한 요청 → 2번 토큰 발급 → 3번 메시지 수신 확인
